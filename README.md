# VLSI CPU Project

This project was developed as part of the VLSI Systems course at the School of Electrical Engineering, University of Belgrade. 
The design was synthesized and deployed on a Cyclone III FPGA development board, where the custom CPU was tested in real hardware.

## Overview

The project focuses on the design and implementation of digital systems using Verilog and SystemVerilog. It covers key concepts in hardware design, including:

- Combinational logic
- Sequential logic
- Parameterized modules
- Finite State Machines (FSMs)
- Simulation and verification
- Logic synthesis
- FPGA implementation

The final stage of the project involved designing and implementing a functional CPU with a custom instruction set.

## Technologies Used

- Verilog
- SystemVerilog
- FPGA development tools
- Simulation and synthesis tools
